目标：开箱即用的一套bootstrap UI管理后台，自带成熟的操作权限管理，让开发专注于后台业务功能开发\<br>
\<br>
架构：spring boot2 + spring security + mybatis + mybatis plus\<br>
 UI : bootstrap + custom jquery plugin\<br>
